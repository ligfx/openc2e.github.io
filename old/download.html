alpha 0.1 "peril" - May 1st, 2008

This is the first public release of openc2e. It is intended primarily as a preview of things to come- it is not by any means feature complete yet. The main focus of this release was Creatures 1, and as a result, of the several games you can expect it to generally work the best. However, please do not expect all of Creatures 2, Creatures 3, and Docking Station to work, and do not bother reporting bugs relating to these games yet (and in particular, relating to the creatures themselves) as we are well aware that many features of those games are broken or currently lacking.

Downloads:

  • Source package: openc2e-0.1.tar.gz
  • Windows (2000/XP/Vista): openc2e-0.1-win32.exe - This installer will automatically locate and create shortcuts for Creatures games installed on your machine.
  • Ubuntu (Gutsy, Hardy, Intrepid): Refer to the package archive, or see the developers page for instructions on building from source.
  • Mac OS X (Currently 10.5/PowerPC only): openc2e-0.1.dmg - Please see the included readme file for usage instructions. We hope to support Mac OS X 10.4 and hopefully Intel processors in a future release.
